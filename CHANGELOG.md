## 当前完成状态

## 完成版功能清单

### 游戏流程

- 开始菜单进入游戏。
- 第 1 层开始挑战，最高到第 30 层。
- 普通敌人、精英、首领和最终首领按楼层出现。
- 战斗胜利后进入奖励选择。
- 指定楼层进入商店。
- 失败、通关和重新开始流程完整。

### 战斗系统

- 攻击、防御和技能牌三类卡牌。
- 点击卡牌后选择目标。
- 攻击牌作用于敌人，防御牌和技能牌作用于玩家。
- 支持暴击、暴击伤害、吸血、护甲、护甲保留、护甲转伤害、反伤和下一次攻击增伤。
- 敌人拥有攻击、强击、防御和蓄力意图。
- 战斗日志记录关键事件。

### 构筑系统

- 起始牌组由 3 张打击和 2 张防御组成。
- 奖励可提升攻击、防御、生命上限、暴击率、暴击伤害、吸血、金币或回复生命。
- 奖励和商店可获得遗物。
- 商店可购买普通或强力卡牌。
- 商店可删除卡牌，帮助玩家压缩牌组。

### 经济和商店

- 击败敌人获得金币。
- 精英、首领和最终首领提供更高金币收益。
- 商店楼层为第 4、9、14、19、24、29 层。
- 商店提供卡牌、遗物、回复生命、最大生命提升和删牌服务。
- 已购买项目会被记录，避免重复购买。

### 记录和部署

- 使用 localStorage 保存最好楼层、最高伤害和通关次数。
- 项目可本地运行、静态构建和 GitHub Pages 部署。
- README 提供 iframe 嵌入示例，可用于主站 LAB 页面展示。

## 项目状态总结

从 2026-06-03 首次上线到 2026-06-06 完成最终视觉修补，《数值高塔》经历了四个主要阶段：

1. 首版上线：完成基础战斗、奖励、记录、部署和文档。
2. 系统扩展：加入金币、商店、牌组、删牌和遗物购买，形成成长闭环。
3. 战斗深化：重做护甲、防御和敌人意图，并把操作区升级成手牌与目标选择。
4. 体验完成：加入开始菜单、历史记录展示、启动 Logo 和场景过渡动画。

当前版本已经具备完整可玩的高塔构筑体验，也具备独立部署、GitHub Pages 发布和主站 iframe 嵌入所需的工程结构与文档说明。
